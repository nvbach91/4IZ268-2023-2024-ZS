Currency Converter

  • This app allows users to make currency conversions.
  • There are 11 currencies supported - 10 of the most used currencies in the world and Czech koruna.
  • Same currency / negative amount / null amount conversions are disabled - Users are allowed to input only positive amounts.
  • Only a dot (.) can be used, comma is disabled.
  • In case of a typing mistake (more zeros than necessary) or nonsense amount in the input field, the value is cleared.
