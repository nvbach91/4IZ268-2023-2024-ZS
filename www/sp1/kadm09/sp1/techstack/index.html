Obsidian

Obsidian je můj knowledge managment systém, bez kterého bych dnes nedal ani ránu. Offline first, modulární BE, performance a stability driven platforma, která podporuje několik přístupů k ukládání a práci s poznámkami. Je zdarma, nativně podporuje exporty poznámek, pluginy lze zcela svobodně tvořit v JavaScriptu a vzhled si každý může napsat sám v CSS stylesheetu, co víc si přát?

The Good

  • Offline first - core data jsou vždy uložené jako pure Markdown, configy v .json
  • Skvěle odladěná premium podpora - publikace jako web wiki, full sync support a další
  • Modulární design - Lze plně konfigurovat přes JS a CSS
  • Komunitní ekosystém předbalíčkovaných pluginů
  • Nativní podpora code highlightingu, LateXu, fulltext searche, metadat indexe a graph view
  • Umí SQL-like agregace a queries
  • I na velkých datech pracuje rychle

The Bad and the Ugly

  • Ruční implementace synchornizace přes git chce trochu fištróna, zlobí binárky
  • Bez premium balíčků se velmi špatně řeší synchronizace pro mobily
  • Trpí stejnými neduhy jako Markdown, problematičtější práce s obrázky
  • Některé integrace souborů (např. pdf) nejsou moc user friendly
  • UX má vysoký skill floor, pro non-plaintext puritány může být na začátku špatně uchopitelné
  • Neumí kolaboraci na úrovni source dat

Ultimate IDEA

Netbrains mě jednou pro vždy překlopil z týmu light-weight na tým IDE. Podpora pro docker services, nativní connectory do databází, debugger, rychlá integrace pluginů a podpora pro Kotlin, Maven, JS a JSON out of the box je pro mě dnes esenciální při pracovních výkonech při BE web vývoji.

The Good

  • Class driven UI a UX, zvládne louskat kontexty a scoping napříč celým projektem
  • Vestavěné DB managment systémy jsou fire
  • Nativně chápe kontejnerizovanou strukturu, umí komunikovat s Dockerem out of the box
  • Full-text search napříč celým projektem
  • Umí držet konzistenci napříč celým projektem
  • Autooptimalizace importů a formátování
  • By default odchytá mnoho sémantických erroů a umí hlídat security vulnerability

The Bad and the Ugly

  • RIP RAM, CPU a battery life
  • Na velkých projektech běží velmi, velmi, velmi pomalu
  • Ultimate verze nějaký ty chectáky stojí
  • Občas trochu dělá problémy s Maven setupem

Visual Studio Code

Nehledě na to jak moc mi IDEA přirostla k srdci, pravda je taková, že všechno v ní neodmakám. Na ten zbytek přichází jako zachránce VSC. Skrze pluginy a moudlární design jde ohnout pro téměř libovolný usecase, v mém případě od Ardiuna po XML.

The Good

  • Zvládne v podstatě libovolný jazyk nebo technologii
  • Lightweight install
  • Podpora pro WSL
  • Zvládne i usecases, které typicky vyžadují specializované editory (XML schemas, transformace)
  • Umí i jazyky, pro které se špatně hledá IDE podpora (Rust)
  • Free

The Bad and the Ugly

  • V low RAM situacích bohužel kucká stejně jako fullblown IDEs
  • Podpora OOP jazyků není ideál, rozject Java projekty je vyloženě bolest
  • Trochu zvláštní default nastavení (autosave,project trust systém,otevírání tabs)
  • VCS integrace mi moc nesedí (Atom did it better)
  • Přijde mi, že autocomplete a intellisense není tak dobrý, jako u jiných produktů
